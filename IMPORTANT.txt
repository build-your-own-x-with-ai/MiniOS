  ld: unknown -arch name: armv4t
  
这不是代码问题，而是 macOS 工具链限制。

================================================================================
                            解决方案
================================================================================

选择以下任一方法：

1. 使用 Docker（最简单）
   ./build-docker.sh

2. 安装交叉编译工具链
   brew tap nativeos/i386-elf-toolchain
   brew install i386-elf-gcc
   然后修改 Makefile 使用 i386-elf-gcc

3. 在 Linux 虚拟机中构建
   sudo apt-get install build-essential nasm grub-pc-bin xorriso
   make
